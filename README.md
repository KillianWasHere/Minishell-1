# Minishell-1
An academic project I made during my first year at EPITECH.

The goal is simple : Remake a shell like TCSH.
This shell can :
- Run commands from every location specified in the PATH environment variable
- Set and unset any environment variable
- Display the environment variable
- (Partially) CD in all locations
- Be exited with "exit"
- Can be ran with "echo [insert a command here]"

This project was made alone in 3 weeks.
